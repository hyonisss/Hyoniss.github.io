Intent로 Activity 전환 및 값 전달

오늘은 xml은 생략.

Intent 전환

MainActivity.java


Intent intent = new Intent(MainActivity.this, SecondActivity.class);
Button button = findViewById(R.id.button);
button. setOnClickLinstener(new View.OnClickListener)() {
public void onClick(View view) {
startActivity(intent);
}
});

버튼을 누르면 intent가 수행되고, MainActivity위에 SecondActivity가 로드된다.
setOnClickListenr가 버튼을 누른 뒤 실행되는 코드를 써주면 되는 곳이다.

그냥 Activity 전환은 간단한데 값 전달하기가 조금 까다롭게 느껴졌다.

Intent로 값 전달하기

MainActivity.java

MainAcitivy에서 SecondActivity로 간단한 자료형(int, String, Boolean 등)의 데이터를 intent에 포함하여 전달할 수 있다.
인텐트를 정의한 후, putExtra를 이용해서 담을 수 있다. 


public class MainActivity extends AppCompatActivity {
protected void onCreate(Bundle savedInstanceState) {
super.onCreate(savedInstanceState);
setContentView(R.layout.activity_main);

Intent intent = new Intent(MainActivity.this, SecondActivity.class);
intent.putExtra("번호", 12345);
intent.putExtra("메시지", "이것은 메시지입니다");

}
}

여기서 putEatra(Key, Value) 형식으로 담아야 한다.
Key는 항상 String 형태, Value에는 int, String 등 다양하게 올 수 있다.

SecondActivity.java

Intent intent2 = getIntent();
intent2.getIntExtra("번호", 0);
intent2.getStringExtra("메시지");

int의 경우 value에 default 값을 적어준다.

여기서 만약, SecondActivity에 있는 TextView에 MainActivity에서 받은 String값으로 바꾸고 싶다면 다음과 같이 하면 된다.

SecondAcitivity.java

TextView tv = findViewById(R.id.textView);

Intent intent2 = getIntent();
String message = intent2.getStringExtra("메시지");

tv.setText(message);

String 변수를 선언해서 String 값을 받고 setText를 이용해 화면에 출력하는 방식이다.
